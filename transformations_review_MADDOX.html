üìö Detailed Explanations

üìç Translation (Slide)

A translation moves every point of a figure the same distance in the same direction. Think of it as sliding a piece of paper across a table.

Key Points:

  • The figure looks exactly the same, just in a new location
  • Use vectors to describe translations: ‚ü®a, b‚ü© means move a units horizontally and b units vertically
  • Positive a = right, negative a = left
  • Positive b = up, negative b = down

Visual Example:

Triangle ABC with vertices A(1,2), B(3,2), C(2,4)

Translated by ‚ü®4, -3‚ü©

New vertices: A'(5,-1), B'(7,-1), C'(6,1)

ü™û Reflection (Flip)

A reflection flips a figure over a line called the line of reflection. Every point on the original figure has a corresponding point on the reflected figure that is the same distance from the line of reflection.

Common Lines of Reflection:

  • x-axis: (x, y) ‚Üí (x, -y)
  • y-axis: (x, y) ‚Üí (-x, y)
  • y = x: (x, y) ‚Üí (y, x)
  • y = -x: (x, y) ‚Üí (-y, -x)

Memory Tip:

When reflecting over the x-axis, the x-coordinate stays the same, but y becomes its opposite!

üîÑ Rotation (Turn)

A rotation turns a figure around a fixed point called the center of rotation. The most common center is the origin (0,0), and rotations are usually 90¬∞, 180¬∞, or 270¬∞.

Standard Rotations about Origin:

  • 90¬∞ counterclockwise: (x, y) ‚Üí (-y, x)
  • 180¬∞: (x, y) ‚Üí (-x, -y)
  • 270¬∞ counterclockwise: (x, y) ‚Üí (y, -x)

Memory Trick for 90¬∞ CCW:

"Negative y comes first, then x" ‚Üí (-y, x)

üîç Dilation (Resize)

A dilation changes the size of a figure by multiplying all coordinates by a scale factor. The center of dilation is usually the origin unless specified otherwise.

Scale Factor Rules:

  • k > 1: Enlargement (figure gets bigger)
  • k = 1: No change
  • 0 < k < 1: Reduction (figure gets smaller)
  • k < 0: Changes size AND orientation

Example:

Point (6, 8) dilated by scale factor 0.5 ‚Üí (3, 4)

The figure is half the original size!

üìù Key Takeaways

Remember These Important Points:

  • Rigid Transformations: Translation, Reflection, and Rotation preserve size and shape
  • Non-Rigid Transformation: Dilation changes size but preserves shape
  • Congruent vs Similar: Rigid transformations create congruent figures; dilations create similar figures
  • Order Matters: When combining transformations, the order can affect the final result
  • Always Check Your Work: Plot points when in doubt to visualize the transformation
